cd (디렉토리명) => 디렉토리명으로 이동
ls => 현재 디렉토리의 파일들 나열
ll => 현재 디렉토리의 파일들 나열하는데 파일, 디렉토리의 권한이나 용량등 더 다양한 정보를 보여줌
ls -al 혹은 ll -al => 숨김 파일까지 포함해서 나열
cd 혹은 cd ~ => 홈으로
pwd => 현재 작업중인 디렉토리 표시

mkdir (디렉토리명) => 디렉토리 생성
mkdir -p (디렉토리명1)/(디렉토리명2)/(디렉토리명3) => 디렉토리 여러개 한번에 생성
touch (파일명) => 빈파일 생성
echo "내용" > (파일명) => 파일에 "내용"을 입력한 채로 생성
rm (파일명) => 파일 삭제
rmdir (디렉토리명) => 빈디렉토리 삭제
rm -r (디렉토리명) => 내용이 있는 디렉토리 삭제
rm -i (파일명) => 삭제하기전 확인요청
rm -rf (디렉토리명) => 디렉토리 강제삭제

cat (파일명) => 파일전체 내용 출력
less 혹은 more (파일명) => 파일의 내용을 페이지별로 나누어서 출력
tail (파일명) => 마지막 10줄만 출력
tail -f (파일명) => 실시간으로 변경되는 내용 계속 출력(로그파일확인에 유용)
head (파일명) => 처음부분 10줄만 출력

vi (파일명) => 파일전체 내용을 출력하며 편집가능
    vi내에서 커맨드
    기본모드 명령어
    - 방향키 혹은 h, j, k, l => 커서를 왼쪽, 아래, 위, 오른쪽으로 움직입니다.
    - w => 다음 단어로 이동
    - b => 이전 단어로 이동
    - 0 => 라인의 시작으로 이동(숫자패드 0으로 치면 바로 작성 돼버림)
    - $ => 라인의 끝으로 이동
    - yy => 현재 라인 복사
    - dd => 현재 라인 삭제
    - p => 복사하거나 삭제한 내용 붙여넣기
    - u => 최근 명령 취소
    - Ctrl + r => 취소된 명령 되돌리기
    - /(검색어) => 앞에서부터 텍스트 검색
    - ?(검색어) => 뒤에서부터 텍스트 검색
        검색이후
        - n => 현재 검색한 방향으로 다음 검색결과
        - N => 현재 검색한 반대 방향으로 다음 검색결과
        이외에 대소문자 구분, 구분없이, 검색결과 강조 같은 기능있음.
    - :%s/old/new/g => 전체 파일에서 'old'를 'new'로 바꾸기
    
    삽입 모드 명령어
    - i => 현재 커서 위치에서 삽입 모드 진입
    - I => 라인의 시작에서 삽입 모드 진입
    - a => 현재 커서 위치의 다음 위치에서 삽입 모드 진입
    - A => 라인의 끝에서 삽입 모드 진입
    - o => 다음 라인에서 삽입 모드 진입
    - O => 이전 라인에서 삽입 모드 진입
    - Esc => 삽입 모드에서 기본 모드로 돌아감
    
    저장 및 종료
    - :w => 파일 저장
    - :q => 에디터 종료
    - :wq 또는 ZZ => 파일을 저장하고 에디터 종료
    - :q! => 변경사항을 저장하지 않고 강제로 종료

chmod (u+r 혹은 u-r) (파일명) => 파일명에대한 소유자의 권한 부여 및 제거
    유저유형
    u => 소유자
    g => 그룹
    o => 다른 사용자
    a => 모든 사용자
    권한
    r => 읽기(4)
    w => 쓰기(2)
    x => 실행(1)
    숫자를 사용해서 권한 부여가능
    chmod 777 (파일명) => 모든 유저에게 파일명에 대한 rwx권한 부여
    chmod 741 (파일명) => 소유자는 rwx, 그룹은 w, 다른유저는 x 권한 부여
chmod -R 777 (디렉토리명) => 디렉토리 및 그 하위항목에 대한 rwx권한 부여
chgrp (그룹명) (파일명) => 파일의 그룹변경
chown (소유자명) (파일명) => 파일의 소유자변경
