서버랙
서버, 스토리지 장치, 네트워크 장비, 통신 장비 등의 데이터 센터 장비를
체계적이고 안정적으로 설치, 저장 및 관리하기 위한 프레임 또는 캐비닛입니다. 
표준화된 크기: 서버랙은 대부분 19인치 폭을 기준으로 설계되며, 
이는 IT 장비의 표준 폭과 일치합니다. 높이는 'U' 또는 'RU' 단위로 측정되며, 1U는 약 1.75인치 (44.45mm)입니다.

IDC(Internet Data Center) 
IDC는 기업이나 기관의 중요한 데이터와 IT 인프라를 중앙 집중화하여 
안전하게 저장, 운영, 관리하는 전문 시설을 의미합니다.


서버 / 라우터 / 방화벽 / HUB / 스위치

서버
서버랙의 주요 구성 요소 중 하나로 
이 서버들은 웹, 데이터베이스, 파일, 응용 프로그램 등의 다양한 서비스를 제공하기 위해 설치됩니다. 
일반적으로 1U, 2U, 4U 등의 표준 크기로 제공되며, 랙 마운트 서버 형태로 설계되어 있습니다.

라우터
네트워크 간의 통신을 가능하게 하는 장비로, IP 주소를 기반으로 데이터 패킷을 적절한 목적지로 전달합니다.

방화벽 (Firewalls)
네트워크의 보안을 강화하는 장비로, 특정 규칙에 따라 데이터 패킷의 전송을 허용하거나 차단합니다.

HUB
물리적 또는 논리적으로 네트워크 세그먼트를 연결하는 기본적인 장비.

스위치
내부 네트워크의 장비 간 통신을 지원하는 장비로, MAC 주소를 기반으로 데이터 프레임을 적절한 포트로 전달


네트워크 연결 흐름
케이블 > 라우터 > 방화벽 > HUB / 스위치 > 서버


공유기
네트워크에서 중요한 장치로, 여러 컴퓨터나 기기들이 하나의 인터넷 연결을 공유할 수 있게 해주는 역할을 합니다.
공유기는 특히 집이나 사무실 같은 환경에서 네트워크 통신의 중심 역할을 합니다.

VPN(Virtual Private Network, 가상사설 네트워크)
인터넷과 같은 공개된 네트워크를 사용하여 원격 위치와 사설 네트워크 간에 안전한 연결을 제공합니다.
비즈니스: 회사의 직원이 외부에서 안전하게 사내 네트워크에 접근할 수 있게 합니다.
개인: 인터넷 사용 시 개인의 프라이버시를 보호하거나 지역 제한을 우회하여 컨텐츠에 접근하는데 사용됩니다.

포트포워딩
네트워크 주소 변환 (NAT)의 일부로서, 
외부 네트워크에서 특정 IP 주소와 포트 번호로 들어오는 트래픽을 
내부 네트워크의 다른 IP 주소와 포트 번호로 전달하는 기능입니다.

집이나 사무실의 내부 네트워크에서 호스팅되는 서버 (웹 서버, 게임 서버 등)에 외부에서 접근할 때 사용됩니다.
일반적으로 방화벽 또는 라우터 뒤에 위치한 서버에 외부 트래픽을 안전하게 전달할 때 필요합니다.
